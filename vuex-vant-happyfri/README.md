# Vuex-vant-happyfri

非常简单的一个 vue2.6 + vuex 的项目，整个流程一目了然，麻雀虽小，五脏俱全，适合作为入门练习，原文地址：[Github](https://github.com/bailicangdu/vue2-happyfri.git)。

```js
{
  "dependencies": {
    "vant": "^2.8.4",
    "vue": "^2.6.11",
    "vue-router": "^3.2.0",
    "vuex": "^3.4.0"
  }
}
```

## Project setup

```sh
git clone ...
cd vuex-vant-happyfri
npm install
npm run serve
```
